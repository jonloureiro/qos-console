# QoS Console

## How to use

#### Install Netlify CLI

```bash
npm install netlify-cli -g
```

#### Create .env file

Example in .env-sample file.
DynamoDB credentials required.
Public and private keys are generated by generateKey.js script.

```bash
node generateKey.js
```

#### How run local server

```bash
npm run dev
```
